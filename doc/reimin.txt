*reimin.txt*  Plugin for adding #includes

Matan Nassau <matan.nassau@gmail.com>

#include a header with less distraction.


:Include                                            *reimin-add-include*

Add an #include after the last #include line above the current position of the
cursor. You will be asked to enter the header to #include on the command line.
You may optionally precede the header name with < character to specify a system
header, or a " to specify a local header.
